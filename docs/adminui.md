# Admin UI
Jeder FreeRP Server hat eine eigene UI mit der man den Server administrieren kann. 
Ein Passwort ist nicht festgelegt. Man sollte aber unbedingt nach dem ersten Start eins setzen. 
Das Passwort mit weiteren wichtigen Einstellungen wird in frpdata/frpsettings.json gespeichert. 
Nach Änderungen in der frpsettings.json muss die Anwendung neu gestartet werden, damit diese wirksam werden.
